# ToDo

本文件记录当前项目需要完成的任务列表。

## 说明

- 每条 ToDo item 使用 Markdown checkbox 格式：`- [ ]` 表示未完成，`- [x]` 表示已完成
- AI（Todo Worker）被触发时，会从未完成的任务中选择一项执行，完成后将其标记为 `- [x]`
- 任务描述应清晰、具体，包含足够的上下文让执行者理解需求
- 可以使用缩进来添加子任务或补充说明

## 格式示例

```markdown
- [ ] 实现用户登录功能
  - 支持邮箱和手机号登录
  - 添加登录失败重试限制
- [x] 修复首页加载缓慢的问题
- [ ] 添加单元测试覆盖 utils 模块
```
