# Workbook — 2026-04-19

## SN: Avatar AmbientDriver 原语 + 默认 breath driver

### Problem
纯 Cubism preview renderer 关掉了 pixi-live2d-display 内置的 breath / eyeBlink / idle motion，期望 bot 全权驱动。但 bot 大部分 idle tick 发 `params: {}` 空帧，renderer 每帧 reset 到默认态，模型视觉凝固。

### Solution
在 AnimationCompiler 里引入 `AmbientDriver` 原语（连续正弦驱动），与既有 discrete action 并行。新增 `DEFAULT_AMBIENT_DRIVERS` 一组默认 driver（baseline-breath，4 channels 对标 Cubism SDK 4 native breath）。通过新增 `compiler.ambientDrivers.enabled` opt-in 启用。Driver 贡献与 action 贡献每 tick **additively mix**。Driver 有 `gate(state)` 按 BotState 全局缩放（speaking 0.3，让 LLM action 更突出）。
