## SN: Avatar Preview Server

### Background

Local preview + LAN streaming enables developers to observe Live2D avatar parameter updates and animation state in real-time from any device on the same network. The bot runs on a server (e.g., a NAS or development machine); the preview page can be opened on a separate laptop or phone connected to the same LAN — no browser extension or USB cable required.

### What was implemented (`src/avatar/preview/**`)

- **`PreviewServer.ts`** — Bun.serve-based HTTP + WebSocket server with `idleTimeout=255`. Serves `GET /` (self-contained HTML preview page) and `GET /health` (JSON with client count). Maintains a `Set` of connected WebSocket clients and broadcasts `frame` and `status` messages to all of them. Caches the latest `PreviewStatus` and sends it immediately to new clients on connect.
- **`types.ts`** — `PreviewConfig`, `PreviewFrame`, `PreviewStatus`, `FrameMessage`, `StatusMessage`, and `PreviewMessage` union type.
- **`index.ts`** — Public re-export barrel.
- **`client/index.html`** — Self-contained preview page. Shows connection state, status metrics (state, fps, activeAnimations, queueLength), and sorted params with progress bars. Derives its WebSocket host from `location.hostname` so it works on LAN.

### WebSocket message shapes

**`frame`** — broadcast on every animation tick with current parameters:
```json
{
  "type": "frame",
  "data": {
    "timestamp": 1713907200000,
    "params": {
      "ParamAngleX": -12.5,
      "ParamAngleY": 8.3,
      "ParamBodyAngleX": -1.2
    }
  }
}
```

**`status`** — broadcast on state changes and periodically:
```json
{
  "type": "status",
  "data": {
    "state": "idle",
    "fps": 45.2,
    "activeAnimations": 2,
    "queueLength": 0
  }
}
```

### LAN configuration

PreviewServer binds to `0.0.0.0:8002` (configurable via `PreviewConfig.host`/`port`). The preview page derives its WebSocket URL from `location.hostname` (same hostname as the page itself), so it automatically connects to the correct LAN IP when opened from another machine.
