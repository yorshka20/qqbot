请为当前群生成一份今日群聊每日汇报。

## 执行步骤

### 第一步：获取今日聊天记录

调用 fetch_history_by_time 工具获取从今天 00:00 到当前时间的所有聊天记录：
- startTime: "00:00"
- includeBot: false

### 第二步：分析聊天内容

仔细阅读所有消息，分析以下内容：

1. **统计数据**：总消息数、活跃成员数
2. **每小时活跃度**：统计每个小时的消息数量（0-23点），用于绘制活跃度图表
3. **最活跃时段**：找出消息最密集的时间段（如 "20:00-22:00"）
4. **今日话题**：提炼 2-4 个主要讨论话题，每个话题写一句概括
5. **成员点评**：选出 3-5 位最活跃或最有趣的成员，为每人写一段幽默有趣的点评（不要太长，1-2句话）
6. **精选发言**：挑选 2-4 条最有趣、最有代表性、或最有深度的发言，附上你的评价（为什么选这条）
7. **群聊总评**：用 2-3 句话总结今天群聊的整体氛围和亮点

### 第三步：组装报告数据并渲染

将分析结果组装为 JSON，调用 render_group_report 工具。reportData 格式如下：

```json
{
  "groupName": "群名称",
  "date": "2024-01-15",
  "totalMessages": 128,
  "activeMembers": 15,
  "highlightTimeRange": "20:00-22:00",
  "hourlyActivity": [
    {"hour": 0, "count": 2},
    {"hour": 1, "count": 0},
    ...（0-23小时每小时一条）
  ],
  "topics": [
    {"title": "话题标题", "summary": "简要概括"},
    ...
  ],
  "memberHighlights": [
    {"userId": "12345", "nickname": "昵称", "messageCount": 30, "comment": "幽默点评"},
    ...
  ],
  "featuredMessages": [
    {"userId": "12345", "nickname": "昵称", "content": "原始发言内容", "comment": "为什么选这条+评价"},
    ...
  ],
  "totalSummary": "今天的群聊总评..."
}
```

注意：
- userId 必须使用聊天记录中真实的用户ID（纯数字字符串）
- nickname 使用聊天记录中的昵称
- hourlyActivity 必须包含 0-23 小时所有数据，没有消息的小时 count 设为 0
- featuredMessages 中的 content 截取原文最精华的部分（不超过80字）
- 点评和总评要有趣、有洞察力

### 第四步：完成

render_group_report 会自动将图片发送到群内。你无需再发送额外消息。
如果报告已成功发送，请简短回复"每日汇报已生成"。
