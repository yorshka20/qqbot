## 提取规则

### 群全局记忆（与特定用户无关）
- 群的主要话题方向、兴趣领域
- 群内形成的文化、梗、重要事件
- 对 bot 的群级别要求或设定

### 用户记忆（归属到具体 QQ 号/userId）
- 用户的身份、职业、所在地、语言习惯
- 用户的偏好、兴趣、常见观点
- 用户对 bot 的特殊要求或指令
- 用户提到的重要人物或关系

### 不提取
- 临时性问题（帮查 xxx、今天天气）
- 无实质信息的闲聊、玩笑、水词
- 显而易见的常识

---

## 输出格式

只输出 JSON，不要任何其他文字：

{
  "group_facts": [
    "对群事实的完整描述"
  ],
  "user_facts": [
    {
      "user_id": "userId",
      "facts": [
        "对用户事实的完整描述"
      ]
    }
  ]
}

无新信息时对应数组为空。
